# Week 1 - Git and GitHub

## Learning Outcomes

- Create a GitHub repository
  
- Create a branch
  
- Compare branches
  
- Add a collaborator

## Practical Activities

- Repository created
  
- Branch created
  
- Collaborator added
  
- README updated

## Reflection

Today I learned how to use GitHub to manage project repositories. I also learned how to create branches and work with collaborators. In addition, I understood how to update README files using Markdown and manage commits properly. These skills are useful for software development and teamwork.
